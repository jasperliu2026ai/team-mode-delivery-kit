# 团队健康度自检清单

> 每周或每月对照检查，1分钟完成

---

## 民主文化健康度

- [ ] 最近是否有角色对老板提议表达过保留意见？
- [ ] 最近的投票是否出现过非全票通过的情况？
- [ ] 阿审是否曾叫停并要求重新讨论？

❌ 全部打叉 → 民主文化失效，参考 `anti-patterns.md` 校准

---

## 交付质量健康度

- [ ] 最近任务老板打分是否≥3分？
- [ ] 是否有任务"说完成了但老板没收到"的情况？
- [ ] 返工率是否连续上升？

❌ 任意打叉 → 交付流程需要检查

---

## 团队纪律健康度

- [ ] 长任务期间是否保持了进度汇报？
- [ ] 阿干是否有未经对齐直接大幅执行的情况？
- [ ] 规矩9条是否有被违反的记录？

---

## 绩效数据健康度

- [ ] 本周是否完成了打分记录？
- [ ] 是否有角色连续两次低于3分？
- [ ] 上月改进动作是否有跟进验收？

---

## 总体判断

- 所有项目通过 → ✅ 团队健康，继续保持
- 1-2项问题 → ⚠️ 局部调整
- 3项以上问题 → 🔴 需要召开复盘会议
