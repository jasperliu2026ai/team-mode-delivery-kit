# 老板使用手册

> 5分钟学会怎么用这个团队

---

## 你的角色

你是**老板**——提需求、排优先级、最终拍板。
产品细节、执行步骤、文档输出，团队包了。你只需要说"要什么"。

---

## 基本操作

### 布置任务
直接说目标，不用说怎么做：
> ✅ "帮我整理这份会议记录，输出行动项"
> ✅ "查一下竞品A和竞品B的差异"
> ❌ 不用说"先让阿查查，然后让阿干写"——阿总会安排

### 打分
任务完成后给个分（1-5），一个字就行：
> "4" / "3分" / "很好5分"

阿审会记录，阿总会复盘。

### 发起投票
说"我们投票一下"，然后说议题，老板你先投：
> "军事化管理需要吗？我选不需要"

### 查看进度
说"给我看看看板"，阿干出截图。

### @单个角色
可以直接点名：
> "@阿查 帮我查一下XXX"
> "@阿干 把这个文件格式改一下"

---

## 团队文化须知

**民主投票是真的**
团队可以投票否决你的提议。这不是不尊重，是团队健康的体现。

**他们会说真话**
如果你的方案有问题，阿查会指出，阿审会打回。这是规矩第9条。

**任务没到你手里不算完**
团队的完成标准是你收到东西，不是"我们做完了"。

---

## 判断团队是否健康

✅ 健康信号：
- 偶尔有角色提出不同意见
- 阿审有时候打回阿干
- 投票不是每次全票通过

⚠️ 不健康信号：
- 所有提案全票通过
- 从没有角色说"我有保留意见"
- 任务完成了但你没拿到东西

---

## 常用指令速查

| 你说 | 效果 |
|------|------|
| 进入团队模式 | 启动团队，开场仪式 |
| @所有人 集合 | 全员报到 |
| 开个会 | 团队会议模式 |
| 给我看看看板 | 截图发任务看板 |
| 投票：XXX | 发起民主投票 |
| 散会 | 结束会议 |
| 记下来 | 阿总存入长期记忆 |
